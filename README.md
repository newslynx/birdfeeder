birdfeeder
======
_A newslynx-opinionated wrapper around twython_

## Install
```
mkvirtualenv birdfeeder
git clone https://github.com/newslynx/birdfeeder.git
cd birdfeeder
pip install -r requirements.txt
pip install .
```

## Test
Requires `nose`
```
nosetests
```
